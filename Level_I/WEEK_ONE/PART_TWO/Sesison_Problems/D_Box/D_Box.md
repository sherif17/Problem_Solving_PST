# Problem Statement
We have removed 
**A**
 balls from a box that contained 
**N**
 balls and then put 
**B**
 new balls into that box. 
 
 How many balls does the box contain now?

## Constraints
-----

All values in input are integers.
* 100
≤
N
≤
200

* 1
≤
A
,
B
≤
100

## Input
------
Input is given from Standard Input in the following format:

**N A B**

### Sample Input 1
----
> **100 1 2**

### Sample Output  1
----
> **101**

**The box contained 
100
 balls. After removing 
1
 ball and putting 
2
 new balls, it now contains 
101
 balls.**
### Sample Input 2
----
> **100 2 1**

### Sample Output  2
----
> **99**


### Sample Input 3
----
> **100 1 1**

### Sample Output  3
----
> **100**
